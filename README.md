init# 🚀 Smart Quote Generator – CI/CD Project (With Lambda)

## Includes:
- Python Application
- Dockerfile
- GitHub Actions CI/CD
- AWS Lambda Function Code
- SNS Notification Setup

## Flow:
Push Code → Build Docker → Push Image → Invoke Lambda → SNS Sends SMS

⚠️ Replace YOUR_SNS_TOPIC_ARN inside lambda_function.py before deploying Lambda.
